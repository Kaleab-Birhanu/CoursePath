# course path

A student and admin course management app which specifically designed for having an easy and organized way of adding and droping of courses.

It's main features are:
  - letting students add courses based on their eligibility and let them to drop courses they don't want to take.
  - a GPA calculator, so that students calculate their grades easily.
  - displaying student's assigned class with their schedule and instructor name.
  - letting admins to recieve the add and drop request, and allow them to assign students in classes.
  - displaying an alert notification when the add and drop deadline is near.
